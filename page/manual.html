Acerca de

Tecnología utilizada

HTML: se utilizó html para marquetación de cada una de las páginas.

CSS: se utilizó css para darle estilo a cada una de las páginas.

Javascript: se utilizó para toda la funcionalidad del videojuego.

Estructura de carpetas

assets: recursos que se utilizan en la página (plugins, archivos, imágenes, etc).

page: aquí se encuentran las páginas que conforman nuestro sitio web.

styles: se encuentran las hojas de estilo que le dan estilo a nuestra página.

Páginas

Index: es la página principal aquí encontrarás mi Hoja de Vida y una tabla correspondiente a mis notas del primer y segundo parcial del presente semestre.

Galeria: en esta página encontrarás una serie de imágenes de mis animes favoritos.

Video: en esta página encontrarás un video de Youtube cargado mediante un iframe.

Videojuego: en esta página encontrarás un videojuego desarrollado mediante html, css y javascript.

Manual: encontrarás información técnica de como se construyó el sitio.

Slider de imágenes

Para crear un carrousel se desarrolló una pequeña librería con css y js. Para poder utilizarla es necesario linkear el siguiente archivo


Dentro del head de la pagina: <link href="../assets/slider/slider.css" rel="stylesheet"/>

Antes de la etiqueta de cierre del body: <script src="../assets/slider/slider.js"></script>


Luego de esto procedos en nuestro html a crear la siguiente estrcutura:

<div class="slider" id="slider"></div>

Y por último colocamos cada imagen de la siguiente manera:

<div class="slide"><img src="../assets/img/1.jpeg" alt="imagen no cargada"> </div>

Snake game

Para utilizar el videojuego es necesario seguir los siguientes pasos


Dentro del head de la página: <link href="../assets/game/game.css" rel="stylesheet" />

Antes de la etiqueta de cierre del body:

<script src="../assets/game/game.js"></script>


Por último solo creamos la seguuiente estructura

<div id="board"></div>

<div class="boardInfo">

     <button id="start"> Start </button>

     <div class="score">

          Score: <span id="scoreBoard"></span>

     </div>

</div>

<div id="gameOver"> GAME OVER</div>
